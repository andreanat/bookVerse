# BookVerse API

BookVerse is a school library management API inspired by the Academia Juárez library.

## Features

- CRUD operations for four collections
- MongoDB Atlas database
- GitHub OAuth authentication
- Protected routes
- Swagger API documentation
- Unit testing for GET routes
- Render deployment

## Collections

- Books
- Authors
- Students
- Checkouts

## Tech Stack

- Node.js
- Express
- MongoDB
- Swagger
- Passport GitHub OAuth
- Jest
- Supertest
- Render
