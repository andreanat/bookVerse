# BookVerse API

BookVerse is a school library management API inspired by the Academia Juárez library.

## Features

* Manage books
* Manage authors
* GitHub OAuth authentication
* MongoDB Atlas integration
* Swagger API documentation
* Render deployment

## Collections

### Books

* title
* author
* genre
* pages
* publishedYear
* language
* available

### Authors

* firstName
* lastName
* nationality
* birthYear
* notableWork

## Technologies

* Node.js
* Express
* MongoDB Atlas
* Swagger UI
* GitHub OAuth
* Render
